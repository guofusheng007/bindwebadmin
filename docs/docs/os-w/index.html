简介

简述​

os-w是基于pxe/ipxe协议而集成的系统，配合我们独立开发的后台(go语言开发)，基于内部网络完成企业内部的标准化操作，可以有效减少IT运维人员的日常工作，规范企业内部标准化管理。。os-w系统实现EFI和BIOS的自动切换，一键批量快速安装各类主流操作系统。无论是Windows、Linux、Ubuntu、Debian、还是ESXi，我们都能轻松应对。更重要的是，我们还为您完成各类配置，让您无需担心繁琐的设置步骤，如部署window系统时可以自动安装各类硬件驱动、常用软件offfice、并自动激活、加域等等。

选择OS-W，让我们的专业团队帮您解决所有IT问题，让您的工作更加高效，生活更加便捷。OS-W，您的IT解决方案专家！

拒绝本地操作、拒绝本地USB接口、拒绝手工介入。

项目参数
整机形态2U机架式服务器
处理器最多 2 个第三代英特尔®至强®可扩展处理器铂金系列，最多 40 个内核，最高可达 270W TDP
内存32内存插槽(8通道/CPU，16 DINMM/CPU) 支持DDR4 3200/2933 RDIMM/LRDIMM/3DS RDIMM，最大支持8TB内存容量。
标配置：512GB内存
存储盘位前置12 *3.5"热插拔盘位
标配置：数据存储 3 * 2T(Raid5),系统盘:2 * 1T(Raid1)
RAID支持RAID级别包括但不限于: 0、1、5、6、10、50 和 60
扩展槽最大支持9个PCle4.0标准插槽
集成芯片集成显示芯片、BMC芯片、SATA控制器，可选集成网络控制器
板载网络1个IPMI专用管理网口，板载四千兆电口，板载双万兆电口。PCIe 适配器
外设接口前置2个USB3口， 后置2个USB3口，1个VGA口
电源双冗余电源（最高 2200W Platinum），支持336V、-48V高压直流电源

功能简述​

  1. 标准化批量部署各类主流系统Redhat/Centos/Rocky/Ubuntu/Debian/ESXi/Windows等，支持主流X86服务器和PC。
  2. 针对windows安装，自动安装各类主流硬件驱动、支持预装应用软件(如office、outlook等)、自动加入域、自动激活。
  3. 支持自定义模板。
  4. 一键备份（某个分区或整个系统）到远程存储、从远程存储一键还原到本地。
  5. EFI/BIOS双模式自动切换。
  6. 集成各类运维工具，满足linux/windows常规运维需求。
